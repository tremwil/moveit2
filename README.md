# `moveit2`

A library for safe, in-place construction of Rust (and C++!) objects.

The [original `moveit` repository](https://github.com/google/moveit) has been archived; this is a maintained fork with some additional features.

# Maintenance Model

While I will take the burden of fixing bugs, I will not take on development for any feature requests
that I do not have a personal use for. I *may* review and merge PRs for them, however.
